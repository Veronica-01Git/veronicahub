# Veronica Hub

Site oficial da **Veronica Hub** — ecossistema de cursos e ferramentas de IA para criadores digitais.

**Produção:** https://veronicahub.com

## Sobre

Plataforma que reúne cursos de IA e o **Veronica Video AI**, motor de geração de vídeo por inteligência artificial, num único hub.

## Stack

Site estático, sem build step nem dependências:

- HTML5 semântico
- CSS puro (variáveis OKLCH, glassmorphism, animações)
- JavaScript vanilla
- Fontes: Bricolage Grotesque, Space Grotesk, JetBrains Mono

## Estrutura

- `index.html` — página principal (cursos + Video AI)
- `index-antigo.html` — versão anterior (backup)
- `video-ai.html` — página do Veronica Video AI
- `veronicacyborgv2.webp` — imagem da Veronica
- `CNAME` — domínio customizado do GitHub Pages

## Deploy

Publicado via GitHub Pages. O arquivo CNAME aponta o domínio veronicahub.com para o repositório.

Todo push na branch `main` publica automaticamente em produção. Alterações devem ser feitas em branch separada e revisadas via Pull Request antes do merge.

## Desenvolvimento local

Não requer instalação. Basta abrir o index.html no navegador, ou servir a pasta com `python3 -m http.server 8000` e acessar http://localhost:8000
